Vad är CSS? Hur används det?

CSS (Cascading Style Sheets) är ett stilarksspråk som används för att beskriva hur HTML-element ska visas på webbsidor. Det används för att definiera layout, färg, typsnitt, storlek och andra visuella egenskaper på webbsidor.


Vad är skillnaden mellan inline-stil, inbäddad stil och extern stil i CSS?

Skillnaden mellan inline-stil, inbäddad stil och extern stil i CSS: Inline-stil: Stilar definieras direkt inuti HTML-elementets style-attribut, och de gäller endast för det specifika elementet. Inbäddad stil: Stilar definieras inuti style-elementet i HTML-dokumentets head. De påverkar alla element på sidan och kan använda selektorer för att välja specifika element. Extern stil: Stilar definieras i en separat CSS-fil och länkas till HTML-dokumentet med en link-tagg. De kan användas på flera sidor och hjälper till att separera innehåll och presentation.


Vad är en selektor i CSS och hur används den?

En selektor i CSS är ett sätt att välja ett eller flera HTML-element som ska stylas. Selektorer kan vara elementselektorer (t.ex. p för stycke-element), klassselektorer (t.ex. .my-class för element med en viss klass), ID-selektorer (t.ex. #my-id för element med ett visst ID) och många andra typer av selektorer. De används för att rikta in sig på specifika element i HTML och tillämpa stilar på dem.


Vad är skillnaden mellan klass-selektor och ID-selektor i CSS?

Skillnaden mellan klassselektor och ID-selektor i CSS: Klassselektor: Används för att rikta in sig på ett eller flera element med en viss klass. Klasser kan återanvändas och tillämpas på flera element i samma dokument. ID-selektor: Används för att rikta in sig på ett enskilt element med ett unikt ID. ID ska vara unika inom samma HTML-dokument och används sällan för att återanvända stilar.


Vad är kaskadordningen i CSS och hur påverkar den hur stilar tillämpas på element?

Kaskadordningen i CSS bestämmer vilka stilar som tillämpas på ett element när flera stilsättningar konflikterar. Det påverkar huruvida en stil från en extern CSS-fil överrider en stil från en inbäddad CSS eller en inline-stil. Kaskadordningen går från mer specifika regler till mer generella regler och kan påverkas av selektorernas specifikitet och ordningen där stilar definieras.


Vad är responsiv webbdesign och hur kan CSS användas för att skapa responsiva layouter?

Responsiv webbdesign innebär att skapa webbsidor som anpassar sig och ser bra ut på olika skärmstorlekar och enheter, inklusive mobiler, surfplattor och datorer. CSS används för att skapa responsiva layouter genom att använda tekniker som mediaförfrågningar (media queries) och flexibla enheter som procent och em för att anpassa storlekar och positioner baserat på skärmstorleken.

Vad är boxmodellen i CSS och vilka element påverkas av den?

Boxmodellen i CSS beskriver hur ett HTML-element är strukturerat med en inre innehållsbox, omgivande ram (border), fyllning (padding) och yttre marginal (margin). Alla dessa element påverkar hur ett element tar upp utrymme på sidan och hur det relaterar till andra element.


Vad är skillnaden mellan positionering med "static", "relative", "absolute" och "fixed" i CSS?

"Grid layout" och "flexbox" är två olika metoder för att skapa layouts i CSS: Grid layout: Använder ett rutnätssystem där element kan placeras i rader och kolumner. Det är idealiskt för att skapa mer komplexa och tvådimensionella layouts. Flexbox: Fokuserar på att organisera element i en enda dimension (rad eller kolumn) och är bra för att hantera innehållsflöden eller centrera element i förhållande till varandra.


Positionering med "static", "relative", "absolute" och "fixed" i CSS:

Static (standard): Elementet positioneras enligt det normala flödet i dokumentet. Relative: Elementet positioneras relativt till sitt normala flöde, vilket möjliggör justering med avstånd från sitt normala läge. Absolute: Elementet positioneras relativt till sitt närmaste överordnade element med icke-statisk position. Det kan användas för att skapa överlappande lager. Fixed: Elementet positioneras relativt till webbläsarens fönster och stannar på samma plats även när sidan rullar. Det används ofta för att skapa sidhuvuden eller sidfotsmenyer.
