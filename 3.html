Vad är HTML och vad används det för?

HTML (Hypertext Markup Language) är ett märkspråk som används för att strukturera och formatera webbsidor. Det används för att definiera och organisera innehållet på en webbsida, inklusive text, bilder, länkar och andra element.

strukturen i ett HTML-dokument

Strukturen i ett HTML-dokument består av följande grundläggande delar: !DOCTYPE: Deklarationen för dokumenttypen, som talar om vilken version av HTML som används. html: Huvudelementet som omsluter hela HTML-dokumentet. head: Innehåller metadata om webbsidan, som titel, länkar till stilmallar och andra information som inte syns på själva sidan. body: Innehåller det synliga innehållet på webbsidan, såsom text, bilder och länkar.

Vad är ett element?

Ett element i HTML är en beståndsdel som representerar en specifik typ av innehåll, och det skrivs med en starttagg och en sluttagg (om det är nödvändigt). Exempel på HTML-element inkluderar p för stycken av text, a för länkar, img för bilder och div för att skapa avsnitt i layouten.

Vad är XML och XHTML?

XML (Extensible Markup Language) är ett märkspråk som används för att definiera egna märkspråk och strukturer för att lagra och överföra data. XHTML (Extensible HyperText Markup Language) är en variant av HTML som är baserad på XML-standarden, och den har striktare regler för kodning och syntaktisk korrekthet.

Vad är DOM, och varför är den viktig?

DOM (Document Object Model) är en representationsmodell som webbläsare använder för att representera och interagera med HTML-dokumentet som en strukturerad trädstruktur. Det är viktigt eftersom det gör det möjligt för webbläsaren och JavaScript att dynamiskt ändra och manipulera innehållet på en webbsida, vilket möjliggör interaktivitet och dynamiska webbapplikationer.
